# Data

There is no external dataset here. The network topology is generated
procedurally by `iotlock.topology.build_topology` using NetworkX's
Barabási-Albert graph generator, and the traffic/failure simulation is
generated by `iotlock.simulation.simulate_cascade`. Both live in
[`src/iotlock/`](../src/iotlock/). See the main README's Data section for
why a Barabási-Albert (scale-free) topology was chosen over a uniformly
random graph.
